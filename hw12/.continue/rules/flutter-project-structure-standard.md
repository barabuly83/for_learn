---

1. main.dart - только точка входа приложения
2. app.dart - корневой виджет приложения (MaterialApp/CupertinoApp)
3. pages/ - экраны/страницы приложения (суффикс _page.dart)
4. widgets/ - переиспользуемые виджеты
5. models/ - модели данных (использовать freezed для immutable моделей)
6. viewmodels/ - бизнес-логика и состояние
7. services/ - сервисы для работы с внешними источниками
8. themes/ - темы приложения
9. constants/ - константы приложения
10. Создавать папки только при необходимости (принцип YAGNI)
